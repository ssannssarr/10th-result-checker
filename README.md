# Automated Result Checker

A simple yet effective automation script built to monitor academic portals for result updates. Instead of manual refreshing, this script handles the polling process autonomously.

## features
* **Automated Polling:** Checks the result link every 5 minutes.
* **Headless Monitoring:** Runs efficiently in the background.
* **Real-time Detection:** Identifies updates by comparing site data/hash changes.

## Tech Stack
* **Language:** Python
* **Environment:** Optimized for Termux / Android / Linux
* **Libraries:** Requests, BeautifulSoup4 (or Selenium/Playwright depending on your build)
